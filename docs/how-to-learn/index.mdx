## 2025总规划
### 2月学习规划
- [ ] 学习 CSS
- [ ] 学习 迭代器、生成器、Promise
- [ ] 学习 JavaScript 高级技巧

阶段目标：
+ 熟练页面布局、高阶样式选择器使用、复杂样式实际应用:动画、内外阴影、单行省略、多行省略、linearGrident、flex快速布局
+ 熟练使用各种迭代器，并分析使用场景，并使用迭代器加闭包思想，实现树形组件
+ 熟悉生成器使用场景
+ 熟练使用Promise，了解Promise原理，实现一个Promise
+ 熟练使用JavaScript高级技巧，如：柯里化、防抖、节流、深拷贝、浅拷贝、继承、原型链、作用域链、闭包、事件委托、事件冒泡、事件捕获、事件循环、宏任务、微任务、异步编程、回调及ES6新特性（数组、对象处理语法糖）

此阶段尤为重要，是后续框架学习的基础，务必扎实掌握。
