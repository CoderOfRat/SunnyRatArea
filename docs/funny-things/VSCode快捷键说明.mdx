# 键盘快捷键（Windows）中文描述
https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf
## 一、通用快捷键

- `Ctrl+Shift+P, F1`：显示命令面板
- `Ctrl+P`：快速打开，转到文件...
- `Ctrl+Shift+N`：新建窗口/实例
- `Ctrl+Shift+W`：关闭窗口/实例
- `Ctrl+,`：用户设置
- `Ctrl+K Ctrl+S`：键盘快捷键

## 二、基本编辑快捷键

- `Ctrl+X`：剪切行（空选择时）
- `Ctrl+C`：复制行（空选择时）
- `Alt+↑/↓`：将行上移/下移
- `Shift+Alt+↓/↑`：复制行上/下
- `Ctrl+Shift+K`：删除行
- `Ctrl+Enter`：在当前行下方插入新行
- `Ctrl+Shift+Enter`：在当前行上方插入新行
- `Ctrl+Shift+\`：跳转到匹配的括号
- `Ctrl+] / [`：缩进/取消缩进行
- `Home / End`：转到行首/行尾
- `Ctrl+Home`：转到文件开头
- `Ctrl+End`：转到文件结尾
- `Ctrl+↑/↓`：滚动行上/下
- `Alt+PgUp / PgDn`：滚动页面上/下
- `Ctrl+Shift+[`：折叠（收起）区域
- `Ctrl+Shift+]`：展开（取消收起）区域
- `Ctrl+K Ctrl+[`：折叠（收起）所有子区域
- `Ctrl+K Ctrl+]`：展开（取消收起）所有子区域
- `Ctrl+K Ctrl+0`：折叠（收起）所有区域
- `Ctrl+K Ctrl+J`：展开（取消收起）所有区域
- `Ctrl+K Ctrl+C`：添加行注释
- `Ctrl+K Ctrl+U`：移除行注释
- `Ctrl+/`：切换行注释
- `Shift+Alt+A`：切换块注释
- `Alt+Z`：切换自动换行

## 三、导航快捷键

- `Ctrl+T`：显示所有符号
- `Ctrl+G`：转到行...
- `Ctrl+P`：转到文件...（与通用快捷键重复，但在此处也适用）
- `Ctrl+Shift+O`：转到符号...
- `Ctrl+Shift+M`：显示问题面板
- `F8`：转到下一个错误或警告
- `Shift+F8`：转到上一个错误或警告
- `Ctrl+Shift+Tab`：导航编辑器组历史记录
- `Alt+← / →`：后退/前进
- `Ctrl+M`：切换Tab移动焦点

## 四、搜索和替换快捷键

- `Ctrl+F`：查找
- `Ctrl+H`：替换
- `F3 / Shift+F3`：查找下一个/上一个
- `Alt+Enter`：选择所有出现的匹配项
- `Ctrl+D`：将下一个查找匹配项添加到选择中
- `Ctrl+K Ctrl+D`：将上一个选择移动到下一个查找匹配项
- `Alt+C / R / W`：切换区分大小写/正则表达式/全字匹配

## 五、多光标和选择快捷键

- `Alt+Click`：插入光标
- `Ctrl+Alt+↑/↓`：在上方/下方插入光标
- `Ctrl+U`：撤销上一个光标操作
- `Shift+Alt+I`：在所选每行的末尾插入光标
- `Ctrl+L`：选择当前行
- `Ctrl+Shift+L`：选择所有出现的当前选择项
- `Ctrl+F2`：选择所有出现的当前单词
- `Shift+Alt+→`：扩展选择
- `Shift+Alt+←`：缩小选择
- `Shift+Alt+(drag mouse)`：列（框）选择
- `Ctrl+Shift+Alt+(arrow key)`：列（框）选择
- `Ctrl+Shift+Alt+PgUp/PgDn`：列（框）选择页面上/下

## 六、富语言编辑快捷键

- `Ctrl+Space, Ctrl+I`：触发建议
- `Ctrl+Shift+Space`：触发参数提示
- `Shift+Alt+F`：格式化文档
- `Ctrl+K Ctrl+F`：格式化选择
- `F12`：转到定义
- `Alt+F12`：快速查看定义
- `Ctrl+K F12`：在侧边栏中打开定义
- `Ctrl+.`：快速修复
- `Shift+F12`：显示引用
- `F2`：重命名符号
- `Ctrl+K Ctrl+X`：修剪尾随空白字符
- `Ctrl+K M`：更改文件语言

## 七、编辑器管理快捷键

- `Ctrl+F4, Ctrl+W`：关闭编辑器
- `Ctrl+K F`：关闭文件夹
- `Ctrl+\`：拆分编辑器
- `Ctrl+1 / 2 / 3`：聚焦到第1、2或3个编辑器组
- `Ctrl+K Ctrl+←/→`：聚焦到上一个/下一个编辑器组
- `Ctrl+Shift+PgUp / PgDn`：将编辑器左移/右移
- `Ctrl+K ←/→`：移动活动编辑器组

## 八、文件管理快捷键

- `Ctrl+N`：新建文件
- `Ctrl+O`：打开文件...
- `Ctrl+S`：保存
- `Ctrl+Shift+S`：另存为...
- `Ctrl+K S`：保存所有
- `Ctrl+F4`：关闭
- `Ctrl+K Ctrl+W`：关闭所有
- `Ctrl+Shift+T`：重新打开已关闭的编辑器
- `Ctrl+K Enter`：保持预览模式编辑器打开
- `Ctrl+Tab`：打开下一个文件
- `Ctrl+Shift+Tab`：打开上一个文件
- `Ctrl+K P`：复制活动文件路径
- `Ctrl+K R`：在资源管理器中显示活动文件
- `Ctrl+K O`：在新窗口/实例中显示活动文件

## 九、编辑器管理快捷键

- `Ctrl+F4, Ctrl+W`：关闭编辑器
- `Ctrl+K F`：关闭文件夹
- `Ctrl+\\`：拆分编辑器
- `Ctrl+1 / 2 / 3`：聚焦到第1、2或3个编辑器组
- `Ctrl+K Ctrl+←/→`：聚焦到上一个/下一个编辑器组
- `Ctrl+Shift+PgUp / PgDn`：将编辑器左移/右移
- `Ctrl+K ←/→`：移动活动编辑器组

## 十、显示快捷键

- `F11`：切换全屏
- `Shift+Alt+0`：切换编辑器布局（水平/垂直）
- `Ctrl+= / -`：放大/缩小
- `Ctrl+B`：切换侧边栏可见性
- `Ctrl+Shift+E`：显示资源管理器 / 切换焦点
- `Ctrl+Shift+F`：显示搜索
- `Ctrl+Shift+G`：显示源代码管理
- `Ctrl+Shift+D`：显示调试
- `Ctrl+Shift+X`：显示扩展
- `Ctrl+Shift+H`：在文件中替换
- `Ctrl+Shift+J`：切换搜索详情
- `Ctrl+Shift+U`：显示输出面板
- `Ctrl+Shift+V`：打开Markdown预览
- `Ctrl+K V`：打开Markdown预览到侧边
- `Ctrl+K Z`：禅模式（按Esc Esc退出）

## 十一、调试快捷键

- `F9`：切换断点
- `F5`：启动/继续
- `Shift+F5`：停止
- `F11 / Shift+F11`：步入/步出
- `F10`：步过
- `Ctrl+K Ctrl+I`：显示悬停信息

## 十二、集成终端快捷键

- ``Ctrl+` ``：显示集成终端
- ``Ctrl+Shift+` ``：创建新终端
- `Ctrl+C`：复制选择
- `Ctrl+V`：粘贴到活动终端
- `Ctrl+↑ / ↓`：向上/向下滚动
- `Shift+PgUp / PgDn`：向上/向下滚动一页
- `Ctrl+Home / End`：滚动到顶部/底部
